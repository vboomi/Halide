This demo is running a Halide image processing routine compiled for x86-64 native client. The source code is here, and it compiles to this assembly: x86_32 x86_64 arm. It's vectorized and parallelized, and on my machine is close to 10 times faster than this low-level C compiled with -O3 (3ms vs 28ms). You can try it yourself below.

If it doesn't load, you probably need to enable NaCl for non-chrome-web-store sources in about://flags and restart chrome.

Move your mouse over the app to create life.
